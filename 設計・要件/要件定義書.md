## 1.基本情報

## 2.プロジェクト概要
| | 実際の機能 |
| - | --- |
| 1 | ユーザーが描いている様子をリアルタイムでライブ配信 |
| 2 | キャンバス上で自由にイラストを描ける |
| 3 | 制限時間は3分で時間内に作品を完成 |
| 4 | 制限時間終了、または書き終えたタイミングでオークションを終了 |
| 5 | バージョン0.5 |
| 6 | アカウント登録 |
| 7 | サイト上で絵を描くことができる |
| 8 | 入札画面で入札いくらと表示が出る |


## 3.システム概要
絵のオークションシステムのECサイト。
オークションに出す商品をサイト上で描き、出品する


## 4.対象ユーザー数・規模感
| 対象 | 詳細 |
| - | --- |
| 利用ユーザー数 | 100人 |
| データ件数 | 商品100件、オークション履歴5000件程度 |
| 利用端末 | PC/スマートフォン/タブレット |




## 5.機能要件
### 機能名
| # | 機能名 | 内容 | 
| - | --- | ---|
| 1 | 出品 | 絵を描ける(制限時間)・色やペンの変更・タイトル・終了時間設定・開始値・落札価格・ |
|2| 入札・購入 |入札時間の管理・終了設定・アプリ内課金(ツイキャスlikeな)課金額上限・吊り上げbot・自動入札|
|3|アカウント登録|ユーザ名・password必須、メールアドレス登録を任意|
|4|取引通知|登録メール、アカウントに入札・落札の通知|
|5|ライブ配信|スーパーチャットor描きながらオークション・閲覧・コメント
|6|配信画面|コメント閲覧(配信者モード)・オークション設定・金額確認
| 7 | 管理画面 | 商品、アカウントの削除 |

## 7.非機能要件
